Tuffix


Official Linux environment for California State University, Fullerton.




The key goal of this effort is to have a unified programming environment for our
introductory courses that is accessible, fosters collaboration, and enables students and
instructors to share working code with one another.

Major Components

  • Ubuntu 20.04
    •   |  
  • Clang Compiler
    •   |  
  • GNU C++ Compiler
    •   |  
  • Atom Text Editor
    •   |  
  • GDB Debugger
    •   |  
  • Atom's DGB-GDB Front End
Additional tools and libraries are also included to support rich programming assignments, and courses aside from 120-121-131.
